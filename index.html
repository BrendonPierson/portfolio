Welcome to My Personal Portfolio Page

I’m glad you decided to stop by! Let’s start with my story.

I'm a former high school biology teacher who loved my students but hated grading their work. So I found a website that would scan scores. Then I taught myself python webscraping to download, curve, and reformat all student grades with the click of a button. This easily saved me an hour a day, and suddenly I was way more interested in refining the code than the student work.

Now I'm at the Nashville Software School, a 6 month program to learn front and backend development from seasoned industry professionals.

piHome Security

A home automation system with mobile-first web interface. The system collects sensor data using two RaspberryPi's running node servers that update a firebase database. The data is then displayed in real time using angular. When an alarm is tripped, a siren plays locally and the home owner is notified by text message.
